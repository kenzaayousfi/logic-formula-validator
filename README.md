# Logic Formula Validator

An OCaml project implementing algorithms for propositional and first-order logic, including formula validation and syllogism analysis.

## Features

- Propositional logic formulas
- First-order logic predicates
- Naive validity checking
- Validity by negation
- Syllogism representation
- Venn diagram generation

## Technologies

- OCaml
- Dune
